1. API란?
  - 사전적 정의 : API(Application Programming Interface)란 응용 프로그램에서 사용할 수 있도록 운영체제나 프로그래밍
		언어가 제공하는 기능을 제어할 수 있도록 만든 인터페이스를 뜻한다. 주로 파일제어, 창제어, 화상처리, 문자제어
		등을 위한 인터페이스를 제공한다.
  - 서비스와 데이터를 교환하는 경우 - 데이터를 보유한 A사로부터 데이터를 받아와서 B사에 데이터를 가공한 것을 제공
	-> 첫번째 방법 : A사로부터 이메일, 웹하드, 외장하드 등의 방실으로 데이터를 받아 가공한 후 같은 방법으로 B사에 제공
			--> 문제점 : A사의 데이터정보가 변경되었을 경우 실시간으로 B사에게 전달할 방법이 없음, 복잡함
	   두번째 방법 : A사의 데이터베이스 접근 권한을 얻은 후 필요한 데이터를 통해 가공한 후 본사의 데이터베이스 접근권한을
			B사에 열어줌
			--> 문제점 : 데이터베이스 접근권한을 열어주기 때문에 보안이슈 생김
	   세번째 방법 : API를 연동하여 필요한 시점에 필요한 정보를 교환
